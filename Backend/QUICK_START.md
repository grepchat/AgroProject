## ❓ Частые вопросы

**Q: Нужно ли запускать отдельный фронтенд-сервер?**  
A: Нет! Теперь всё на одном сервере.

**Q: Почему два порта (8000 и 8080)?**  
A: Теперь используется только порт 8000. Порт 8080 больше не нужен.

**Q: Камера не работает**  
A: Убедитесь, что:
1. Сервер запущен с HTTPS (через `start_https_server.py`)
2. Сайт открыт через `https://` (не `http://`)
3. Вы приняли предупреждение о сертификате в браузере
